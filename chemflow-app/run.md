# Run the App Locally

### Install Dependencies

```bash
npm install
```

### Add Your API Key

Open `server.js` and replace:

```js
YOUR_GEMINI_API_KEY_HERE
```

### Start the Server

```bash
npm start
```

### Local Server URL

```
http://localhost:3000
```
